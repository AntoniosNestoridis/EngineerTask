# Engineering Task - LXFML Parser
## Made in Unity version 6.000.0.25.f1 
If you experience any issues with cloning or downloading the project from the instructions below, please email me at [nestoridis.antonai@gmail.com]


## Instructions to run the application from the standalone Windows build 
***[Requires microsoft Windows OS]***
  1) Clone/download the repository.
  2) Navigate to the _"LEngineeringTask/Build"_ directory
  3) (Optional) If you wish to test the application with different LXFML files than the default one that was provided, navigate to the subdirectory _“LEngineeringTask_Data\StreamingAssets”_  and move any model files there.
  4) Run the _“LEngineeringTask.exe”_ file to launch the application.

 
## Instructions to run the application in the Unity Editor 
***[Potentially minimally requires Unity version 6.000.0.25.f1 or higher]***
  1) Clone the respository 
  2) Use Unity to open the project from the root directory.
  3) Once the project is open, open the scene _"Parser"_ inside the _"00 Scenes"_ folder.
  4) (Optional) If you wish to test the application with different LXFML files than the default one that was provided, drag and drop model files within the directory _"Assets/StreamingAssets"_.
  5) Press play
