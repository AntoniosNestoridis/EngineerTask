--- Made in Unity version 6.000.0.25.f1 ---


-- Instructions to run the application from the standalone Windows build [Requires microsoft Windows OS] 
  1) Clone the repository or download and unzip the provided Build.zip file
  2) Navigate to the "LEngineeringTask/Build" directory
  3) (Optional) If you wish to test the application with different LXFML files than the default one that was provided, navigate to the subdirectory “LEngineeringTask_Data\StreamingAssets”  and move any model files there.
  4) Run the “LEngineeringTask.exe” file to launch the application.

 
-- Instructions to run the application in the Unity Editor [Potentially minimally requires Unity version 6.000.0.25.f1 or higher]
  1) Clone the respository 
  2) Use Unity to open the project from the root directory.
  3) Once the project is open, open the scene "Parser" inside the "00 Scenes" folder.
  4) (Optional) If you wish to test the application with different LXFML files than the default one that was provided, drag and drop model files within the directory "Assets/StreamingAssets".
  5) Press play
